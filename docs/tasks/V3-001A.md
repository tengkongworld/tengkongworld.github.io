# V3-001A Homepage Configuration

## Background

The homepage currently stores collection data directly inside index.html.

This makes homepage content difficult to maintain and inconsistent with the project's configuration-driven architecture.

---

## Goal

Move homepage collection data into an independent data file.

No UI or functional changes.

---

## Scope

Included

- Create data/homepage.json
- Move homepage collection data
- Load homepage data from homepage.json
- Keep homepage rendering unchanged

Excluded

- UI redesign
- CSS changes
- Homepage layout changes
- New homepage features

---

## Validation

- Homepage renders correctly
- No visual differences
- Homepage data comes from homepage.json
- No JavaScript errors

---

## Acceptance Criteria

✓ Homepage looks identical.

✓ Homepage data no longer exists inside index.html.

✓ Homepage reads data from homepage.json.

✓ Future collection changes require editing only homepage.json.
