V2-005 Script Modularization

Goal

Split scripts/sync.py into maintainable modules without changing any existing behavior.

The objective is to improve maintainability, readability and long-term scalability.

⸻

Design Principles

* Behavior must remain unchanged.
* Refactoring must be incremental.
* Each extraction must be independently testable.
* Each extraction should be committed separately.
* sync.py remains the main entry point.

⸻
